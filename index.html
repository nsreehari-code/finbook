Help

Loading Data

Click Load Data File on the start screen and select your JSON database file (e.g. Sarala_Database.json). The data is cached in your browser's localStorage so it persists across page refreshes. You can also click Load Sample Data to explore the app with demo records.

Saving Data

After editing any records, the Save button appears in the header. Click it to download the updated JSON file. This saves your raw data only — computed fields are stripped to keep the file clean. On save, the localStorage cache is cleared and you'll need to reload the file.

Exporting Computed Data

The Export button (next to Save) downloads a JSON file containing all computed data for the currently selected account and financial year. This includes:

  • Income Summary — totals by category (Salary, Foreign, Property, Capital Gains, etc.) with total taxable income, tax relief, and TDS.
  • Capital Gains — each transaction with lot-matched acquisition cost, holding period, LTCG/STCG classification, and gain/loss.
  • Stock Book — holdings as-on date with lot-level breakdown, average cost, and total portfolio value.
  • All Income Rows — unified view of every income entry across all tables.
  • Computed Records — every table's records with all computed fields included.
Export via URL

You can trigger an export directly by adding query parameters to the URL. This requires data to already be in localStorage from a previous session.

?export=<filename>&account=<account>&fy=<fy>
ParameterDescriptionExample
exportOutput filename (required to trigger export; .json appended if missing)Sarala_2025
accountAccount code or name (case-insensitive, optional — defaults to current)Sarala
fyFinancial year (optional — defaults to latest)2024-25

Example: https://<your-site>/finbook/?export=Sarala_FY2425&fy=2024-25&account=Sarala

CLI Export (Node.js)

A standalone script export.js can generate the same computed export from the command line without a browser. Download it and run with Node.js:

node export.js <input.json> <account> <fy> <output.json>

Examples:

node export.js Sarala_Database.json Sarala 2024-25 Sarala_Computed.json
node export.js Sarala_Database.json Hari All Hari_AllYears.json
